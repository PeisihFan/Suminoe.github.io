判断链表是否有环

可以用快指针与慢指针来解决，有环的话快指针必然会追上慢指针。

public boolean findLoopPort(ListNode head){
    if(head==null||head.next==null)
        return false;

    ListNode fast=head;
    ListNode slow=head;

    while(fast!=null&&fast.next!=null){
        fast=fast.next.next;
        slow=slow.next;
        if(fast==slow){
            break;
        }
    }

    if(fast!=slow)
        return false;
    else
        return true;
}

值得注意的是寻找环入口有个方法

假定起点到环入口点的距离为 a，slow 和 fast 的相交点M与环入口点的距离为b，环路的周长为L，当 slow 和 fast 第一次相遇的时候，假定 slow 走了 n 步。那么有：

p1走的路径： a+b ＝ n；
p2走的路径： a+b+kL = 2n； fast 比 slow 多走了k圈环路，总路程是slow的2倍

根据上述公式可以得到 k*L=a+b=n显然，如果从相遇点M开始，slow 再走 n 步的话，还可以再回到相遇点，同时fast从头开始走的话，经过n步，也会达到相遇点M。

显然在这个步骤当中 slow 和 fast 只有前 a 步走的路径不同，所以当 slow 和 fast 再次重合的时候，必然是在链表的环路入口点上。

因此可以这样寻找

按照 fast 每次两步，slow 每次一步的方式走，发现 fast 和 slow 重合，确定了单向链表有环路了。接下来，让fast回到链表的头部，重新走，每次步长不是走2了，而是走1，那么当 fast 和 slow 再次相遇的时候，就是环路的入口了。

寻找链表中点
此外快慢指针还可以用来寻找链表的中点

public ListNode findMindle(ListNode head){
    ListNode slow=head;
    ListNode fast=head.next;
    while(fast!=null&&fast.next!=null){
        slow=slow.next;
        fast=fast.next.next;
    }
    return slow;
}

链表反转

public ListNode listReversed(ListNode head){
   if(head==null||head.next==null){

       return head;
   }
    ListNode p=head;
    ListNode q=head.next;
    while(q.next!=null){
        ListNode t=q.next;
        q.next=head;
        head=q;
        q=t;
    }
    q.next=head;
    p.next=null;

    return q;
}
